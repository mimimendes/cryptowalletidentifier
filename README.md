# 🔍 Crypto Wallet Identifier

Ferramenta web para identificação de redes blockchain a partir de endereços de carteiras de criptomoedas.

## Sobre
O **Crypto Wallet Identifier** permite colar qualquer endereço de carteira e obter instantaneamente a rede blockchain correspondente, com informações técnicas detalhadas sobre o tipo de endereço, modelo de contabilidade e algoritmo criptográfico utilizado.

## Funcionalidades

- Detecção automática de rede por padrão de endereço
- Identificação de subtipos (ex: Taproot, SegWit, Legacy no Bitcoin)
- Alerta para formatos compartilhados entre redes (padrão EVM)
- Histórico das últimas 8 pesquisas, sem duplicatas
- Interface responsiva com tema escuro
- 100% client-side — sem backend, sem rastreamento

## Tecnologias

HTML · CSS · JavaScript puro (sem dependências externas)

## Como usar

Basta acessar a aplicação em https://cryptowalletidentifier.netlify.app/, colar o endereço da carteira no campo indicado e clicar em **Pesquisar**.
